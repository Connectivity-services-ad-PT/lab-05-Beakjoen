# Lab 05 Reports

This directory contains test reports generated from Newman/Postman execution.

## Files

- `newman-lab05-compose.xml` - JUnit XML format report
- `newman-lab05-compose.html` - HTML visual report with charts
- Screenshots and evidence of health checks

## Generate Reports

Run the following command to generate test reports:

```bash
npm run test:compose
```

Or use Makefile:

```bash
make test-compose
```
